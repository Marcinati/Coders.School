## Przykład tablicy

```cpp
std::array<int, 5> a = { 1, 2, 4, 5, 6}; // eq. int a[5] = { 1, 2, 4, 5, 6};
a[0] = 5;			// a == { 5, 2, 4, 5, 6}
a.at(1) = 7;			// a == { 5, 7, 4, 5, 6}
a[4] = a.front();		// a == { 5, 2, 4, 5, 5}
a.fill(5);			// a == { 5, 5, 5, 5, 5}

std::cout << a.size() 		// 5
std::cout << a.max_size() 	// 5

std::array<int, 5> b = { 5, 6, 7, 8, 9};
b.swap(a); // a == { 5, 6, 7, 8, 9} , b == { 5, 5, 5, 5, 5}
```
